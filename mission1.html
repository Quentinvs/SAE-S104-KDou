Mission 1 – Étude des données

Analyse structurelle de la base KDou avec Excel

Objectifs de la mission

L'objectif de cette mission était de prendre en main la base de données KDou sous forme de fichiers Excel, de comprendre la structure des tables et d'identifier les premières anomalies de données.

  • Comprendre le rôle de chaque table
  • Identifier les clés primaires, candidates et étrangères
  • Étudier les relations entre les tables
  • Repérer les incohérences et problèmes de saisie

Analyse des principales tables

Table Fournisseur

  • Clé primaire : N° fournisseur
  • Clés candidates : Nom, Contact
  • Rôle : identifier les fournisseurs de KDou

Table Produit

  • Clé primaire : Réf produit
  • Clés étrangères : N° fournisseur, Code catégorie
  • Rôle : décrire le catalogue de produits

Table Catégorie

  • Clé primaire : Code catégorie
  • Clé candidate : Nom catégorie
  • Rôle : regrouper les produits et définir des coefficients de marge

Table Clientèle

  • Clé primaire : Code client
  • Clés candidates : Société, Contact
  • Rôle : gérer les clients exclusivement situés en France

Table Commande

  • Clé primaire : N° commande
  • Clé étrangère : Code client
  • Rôle : historiser les commandes passées par les clients

Table DétailCommande

  • Clé primaire composite : (N° commande, Réf produit)
  • Rôle : détailler les lignes de chaque commande

Anomalies détectées

Problèmes de saisie

  • Pays saisis de plusieurs façons : USA / U.S.A, Royaume Uni / Royaume-Uni
  • Villes avec fautes d'orthographe : Besançon / Besancon
  • Formats de téléphone non homogènes
  • Valeurs manquantes (prix, stock, dates...)

Problèmes de conception

  • Colonne Pays inutile dans Clientèle (tous les clients sont en France)
  • Coefficients de marge avec trop de décimales
  • Absence de table normalisée pour les pays

Pistes d'amélioration

  • Nettoyer les valeurs (pays, villes, téléphones...)
  • Compléter ou supprimer les lignes avec données manquantes
  • Normaliser les pays dans une table dédiée
  • Arrondir les coefficients de marge à un format homogène
  • Mettre en place des règles de validation de saisie
