Création du Site Web

R1.02 – Développement d'Interface Web

Présentation du projet

Ce site web a été développé dans le cadre de la ressource R1.02, en complément du rapport de la SAE S1.04. Il rassemble et met en forme les résultats obtenus lors des trois missions sur la base KDou.

Architecture du site

Structure des fichiers

site-web/ │ ├── index.html (Accueil) ├── mission1.html (Mission 1) ├── mission2.html (Mission 2) ├── mission3.html (Mission 3) ├── conclusion.html (Conclusion) ├── creation-site.html (Cette page)

Choix principaux :

  • Un fichier HTML par grande partie de la SAE
  • Un en-tête et un pied de page identiques sur toutes les pages
  • Une navigation fixe centrée en haut de l'écran

Technologies utilisées

HTML5

Utilisation des balises sémantiques (header, nav, main, section, footer).

CSS3

Variables CSS, Flexbox, cartes, tableaux et blocs de code stylisés.

Responsive

Adaptation du site sur ordinateur, tablette et smartphone.

Difficultés rencontrées

Organisation des pages

Problème : garder une navigation cohérente entre plusieurs fichiers HTML.

Solution : factoriser le même en-tête sur toutes les pages.

Responsive

Problème : affichage des cartes et tableaux sur mobile.

Solution : utiliser Flexbox et des media queries.

Lisibilité du code

Problème : garder un code HTML/CSS lisible malgré la quantité de contenu.

Solution : commenter et structurer les sections par thèmes.

Bilan sur le développement web

La réalisation de ce site nous a permis de :

  • Mettre en pratique les bases du HTML5 et du CSS3
  • Comprendre la structure d'un site multi-pages
  • Appliquer une charte graphique cohérente
  • Prendre en compte l'expérience utilisateur (navigation, lisibilité, responsive)
