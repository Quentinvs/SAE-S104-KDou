Introduction

Contexte de l'entreprise KDou

KDou est une entreprise commerciale qui propose à ses clients des produits alimentaires d'épicerie fine ou typiques achetés à des fournisseurs situés dans le monde entier. Tous les clients de l'entreprise se situent actuellement en France. L'entreprise considère générer un assez bon chiffre d'affaires et se pose la question de son développement futur.

Objectifs de la SAE

Dans le cadre de cette SAE, nous avons été amenés à :

  • Analyser et comprendre la structure de la base de données KDou
  • Identifier les clés primaires, clés candidates et clés étrangères
  • Détecter les anomalies et proposer des améliorations
  • Créer des requêtes SQL pour extraire des indicateurs pertinents
  • Exploiter la base de données pour aider à la prise de décision stratégique

Méthodologie adoptée

Notre travail s'est organisé en trois missions complémentaires :

  • Mission 1 : Étude structurelle des données et navigation dans Excel
  • Mission 2 : Exploitation de la base via une interface low code (Access)
  • Mission 3 : Analyse avancée avec SQL pour produire des indicateurs calculés
