Conclusion générale

Synthèse de la SAE S1.04 – KDou

Bilan des outils

Outil Usage Points forts Limites
Excel Exploration brute des données Simple, visuel, filtrage rapide Pas de vrai modèle relationnel, limité pour les analyses complexes
Access Interface low code (QBE) Création de requêtes sans SQL, interface graphique Dépendant de Windows, peu adapté à de très grandes bases
SQL Analyse avancée, indicateurs Très puissant, précis, automatisable Nécessite une vraie maîtrise de la syntaxe

Compétences acquises

Analyse de données

Compréhension d'une base relationnelle, détection d'anomalies, structuration.

Manipulation

Filtres Excel, requêtes Access, écriture de requêtes SQL multi-tables.

Logique

Construction de jointures, sous-requêtes, indicateurs calculés.

Communication

Rédaction de rapport, synthèse, présentation des résultats.

Perspectives pour KDou

Les analyses réalisées donnent à KDou une vision plus claire de son activité : produits performants, clients importants, zones géographiques à développer, et pistes d'optimisation logistique.

  • Automatisation : mettre à jour régulièrement les indicateurs SQL
  • Visualisation : connecter la base à des outils de dataviz (tableaux de bord)
  • Stratégie : ajuster l'offre produits, les marges et la logistique
